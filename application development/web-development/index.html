About the implementations

The implementations were about calculating markers in the vicinity of 1km and calculating the nearest polygon to a clicked point on the map. Both tasks involve user interaction with the map to perform calculations and display results.

Implementations

Map marker calculation

This implementation is about marker visualization. Wherever the user clicks on the map, it calculates how many markers are within a range of 1km. The result is then printed as an alert to the user.

Polygon Calculation

This implementation is about creating a polygon visualization. Wherever the user clicks on the map, the nearest polygon is calculated, highlighted on the map and in the list below.
